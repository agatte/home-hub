System: Single-user, always-on home automation dashboard with ML intelligence
Stack:  FastAPI + SvelteKit + SQLite + WebSocket + LightGBM + YAMNet +
        MediaPipe + Pi-hole (Docker)
Prod:   Dell Latitude 7420 (Ubuntu 24.04) kiosk at 192.168.1.210
Dev:    Windows desktop at 192.168.1.30
Repo:   github.com/agatte/home-hub

This doc tracks the substantial work that has shipped and the substantial
work that has not. Trivial fixes (one-line bug patches, dependency pins,
small a11y nudges) are deliberately omitted -- check git log for those.

This refresh is backed by both static review and a live MCP runtime sample
of the Latitude (April 27, ~11:40pm). Where a finding cites production
data, it's labeled [LIVE].

================================================================================
                          PRIVACY POSTURE
================================================================================

  - Camera frames: numpy arrays in memory only, overwritten each cycle.
    Never serialized, persisted, networked, or logged
  - Audio: raw PCM overwritten each 1s read. Mel-spectrograms one-way.
    Only labels persist
  - Models: aggregate statistical patterns only, no PII reconstructable
  - 90-day rolling retention on event tables
  - Full ML data wipe available from Settings (models + decisions +
    metrics)
  - All inference is local-only -- no cloud APIs
