     - POST /api/scenes/effects/{name} activates dynamic effects

  ✅ Automation engine (automation_engine.py)
     - Per-light control: each light can be set independently per mode
     - Time-aware activity states: brightness adapts to day/evening/night
       Day (8-18): brighter to counteract ambient daylight
       Evening (18-21): moderate brightness
       Night (21-8): dim, easy on eyes
     - Weekday time rules: off at work (7am-6pm), warm evening, dim wind-down
     - Weekend time rules: sleep in, gentle 2hr morning ramp, neutral bright day
     - Activity overrides: gaming, working, watching, social, relax, movie
     - Party sub-modes: color_cycle, club, rave, fire_and_ice
     - Sleep detection: 10:30pm + 15min idle → pause YouTube + fade lights
     - Screen sync: bedroom lamp matches projector content (mss capture)
     - Manual override from dashboard (4-hour auto-timeout, optimistic UI)
     - Alexa coexistence: detects external "all off" (geofence), won't override
     - WebSocket broadcasts mode_update to all clients
     - Background loop checks every 60 seconds, auto-adapts on period transitions

  ✅ PC activity detector (pc_agent/activity_detector.py)
     - Standalone script, runs alongside server
     - psutil process detection every 15 seconds
     - Gaming: LeagueofLegends.exe, javaw.exe (OSRS), 20+ game processes
     - Working: windowsterminal, powershell, code.exe, cursor, devenv, etc.
     - Watching: VLC, Plex, media players
     - Sleep detection: after 10:30pm, 15min idle → pause media + report sleeping
     - Away: Win32 GetLastInputInfo idle >10 min
     - Reports mode changes only to /api/automation/activity

  ✅ Ambient noise monitor (pc_agent/ambient_monitor.py)
     - Blue Yeti microphone via PyAudio
     - Measures RMS volume only — never records or saves audio
     - Party detection: sustained noise >2 min + no game running = "social"
     - No false positives during gaming (headset isolates audio)
     - Auto-calibrates noise floor on startup
     - POST /api/automation/mic/calibrate endpoint

  ✅ Morning routine (morning_routine.py + scheduler.py)
     - Replaces Alexa 6:40 AM weekday routine
     - Fetches weather (OpenWeatherMap API)
     - Fetches commute traffic (Google Maps Directions API)
     - Generates TTS: "Good morning Anthony. It's 45 degrees..."
     - Plays on Sonos, triggers morning light ramp
     - AsyncScheduler: cron-like async scheduling, no external deps
     - POST /api/routines/morning/test for instant testing
     - Configurable time/volume via PUT /api/routines/morning/config

  ✅ Music mapper (music_mapper.py)
     - Maps activity modes to Sonos favorites (Apple Music playlists)
     - User creates playlists per mood, maps in Home Hub
     - Auto-play or suggest on mode change
     - GET /api/sonos/favorites lists available playlists

  ✅ Screen sync service (screen_sync.py)

     - Captures every 2.5s, brightness clamped 15-80

  ✅ Automation API routes (automation.py + routines.py)

API ENDPOINTS (Complete)
================================================================================

  Core:
  GET  /health                              — Service status
  WS   /ws                                  — Real-time state sync

  Lights:
  GET  /api/lights                          — All lights state
  GET  /api/lights/{id}                     — Single light state
  PUT  /api/lights/{id}                     — Update light
  POST /api/lights/all                      — Update all lights

  Scenes & Effects:
  GET  /api/scenes                          — Presets + native bridge scenes
  POST /api/scenes/{id}/activate            — Activate scene (preset or native)
  GET  /api/scenes/effects                  — List dynamic effects
  POST /api/scenes/effects/{name}           — Activate effect on all lights
  POST /api/scenes/effects/{name}/light/{id} — Effect on specific light

  Sonos:
  GET  /api/sonos/status                    — Now-playing info
  POST /api/sonos/play|pause|next|previous  — Playback controls
  POST /api/sonos/volume                    — Set volume
  POST /api/sonos/tts                       — Text-to-speech
  GET  /api/sonos/favorites                 — List Sonos favorites
  POST /api/sonos/favorites/{title}/play    — Play a favorite
  GET  /api/sonos/music-map                 — Mode→playlist mapping
  PUT  /api/sonos/music-map                 — Update a mapping

  Automation:
  POST /api/automation/activity             — Report detected activity
  GET  /api/automation/activity             — Current mode
  GET  /api/automation/status               — Full engine state
  POST /api/automation/override             — Manual mode override
  POST /api/automation/mic/calibrate        — Calibrate mic baseline
  GET  /api/automation/config               — Get automation config
  PUT  /api/automation/config               — Update automation config

  Routines:
  GET  /api/routines                        — List scheduled routines
  POST /api/routines/morning/test           — Test morning routine
  PUT  /api/routines/morning/config         — Update morning config
  POST /api/routines/morning/toggle         — Enable/disable
